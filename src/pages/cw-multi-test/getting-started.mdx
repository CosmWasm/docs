# Getting started

This unit is designed to help you quickly become familiar with the fundamental aspects of testing
CosmWasm smart contracts using **`MultiTest`**. It provides a practical example and best practices
to ensure a frictionless start with **`MultiTest`** for testing your smart contracts.

In the following chapters, you will find a
[smart contract example](getting-started/smart-contract.mdx) for which you will be
[writing tests](getting-started/writing-tests.mdx), along with some
[best practices](getting-started/tests-placement.mdx) for testing with **`MultiTest`**.
