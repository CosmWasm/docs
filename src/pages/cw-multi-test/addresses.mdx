# Addresses

A couple of addresses with the default `cosmwasm` and custom `nebula` prefixes in Bech32 and Bech32m
formats will be used in the examples that follow in this chapter:

```rust showLineNumbers copy
const CW_BECH32: &str = "cosmwasm1fsgzj6t7udv8zhf6zj32mkqhcjcpv52yph5qsdcl0qt94jgdckqs2g053y";
const CW_BECH32M: &str = "cosmwasm1fsgzj6t7udv8zhf6zj32mkqhcjcpv52yph5qsdcl0qt94jgdckqsl5lc5x";
const NB_PREFIX: &str = "nebula";
const NB_BECH32: &str = "nebula1fsgzj6t7udv8zhf6zj32mkqhcjcpv52yph5qsdcl0qt94jgdckqsvsqrvp";
const NB_BECH32M: &str = "nebula1fsgzj6t7udv8zhf6zj32mkqhcjcpv52yph5qsdcl0qt94jgdckqsevs0fr";
```

## App

`cw_multi_test::App`

```rust {1} showLineNumbers copy /api().addr_make/ /assert_eq!(CW_BECH32/
use cw_multi_test::App;

let app = App::default();

let addr = app.api().addr_make("owner");

assert_eq!(CW_BECH32, addr.as_str());
```

```rust {1} showLineNumbers copy /api().addr_make/ /MockApiBech32::new(NB_PREFIX)/ /assert_eq!(NB_BECH32/
use cw_multi_test::{no_init, AppBuilder, MockApiBech32};

let app = AppBuilder::default()
    .with_api(MockApiBech32::new(NB_PREFIX))
    .build(no_init);

let addr = app.api().addr_make("owner");

assert_eq!(NB_BECH32, addr.as_str());
```

```rust {1} showLineNumbers copy /api().addr_make/ /MockApiBech32m::new(NB_PREFIX)/ /assert_eq!(NB_BECH32M/
use cw_multi_test::{no_init, AppBuilder, MockApiBech32m};

let app = AppBuilder::default()
    .with_api(MockApiBech32m::new(NB_PREFIX))
    .build(no_init);

let addr = app.api().addr_make("owner");

assert_eq!(NB_BECH32M, addr.as_str());
```
